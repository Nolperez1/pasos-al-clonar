Pasos Después de Clonar un Proyecto

1. Instalar Composer

  1. Composer install

2. Instalar Dependencias de Node

  1. npm install
  2. Corregir errores si hubiera en la instalación de npm:
    1. npm Audit
    2. npm audit fix

3. Hacer Algunos Ajustes Importantes (Obligatorio)

  1. php artisan storage:link → Crea un directorio de archivos en la carpeta Public
  2. php artisan key:generate → Crea la key única para ese proyecto en APP_KEY=
  3. FILESYSTEM_DISK=local cambiar a PUBLIC
  4. DB_DATABASE=nombre definir el nombre de la base de datos
  5. Crea un dominio local en Windows (ejecuta el Bloc de Notas como administrador) y Apache
  6. Cambiar APP_URL=http://localhost con el nuevo dominio local: APP_URL=http://nuevo.test

4. Hacer las Migraciones

  1. php artisan migrate
  2. php artisan db:seed

5. Inicializar Proyecto de Esta Manera

  1. npm run build

Pasos para subir un proyecto a github

  1. git status -> ver el estado del proyecto
  2. git add . -> agregar todos los archivos
  3. git commit -m "mensaje" -> para guardar los cambios
  4. git push -> para subir el proyecto a github
  5. Esperar un tiempo para ver los cambios en github pages
