Pasos para subir un proyecto a github

  1. git status -> ver el estado del proyecto
  2. git add . -> agregar todos los archivos
  3. git commit -m "mensaje" -> para guardar los cambios
  4. git push -> para subir el proyecto a github
  5. Esperar un tiempo para ver los cambios en github pages
