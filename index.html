Pasos para clonar un proyecto de un repositorio github

    Ir a la ubicacion del proyecto donde va clonarse

  1. git clone https://github.com/TuUsuario/mi-pagina-web.git -> ejecutar este comando
  2. Esperar que se clone

Pasos Después de Clonar un Proyecto de laravel

1. Instalar Composer

  1. Composer install

2. Instalar Dependencias de Node

  1. npm install
  2. Corregir errores si hubiera en la instalación de npm:
    1. npm Audit
    2. npm audit fix

3. Hacer Algunos Ajustes Importantes (Obligatorio)

  1. php artisan storage:link → Crea un directorio de archivos en la carpeta Public
  2. php artisan key:generate → Crea la key única para ese proyecto en APP_KEY=
  3. FILESYSTEM_DISK=local cambiar a PUBLIC
  4. DB_DATABASE=nombre definir el nombre de la base de datos
  5. Crea un dominio local en Windows (ejecuta el Bloc de Notas como administrador) y Apache
  6. Cambiar APP_URL=http://localhost con el nuevo dominio local: APP_URL=http://nuevo.test

4. Hacer las Migraciones

  1. php artisan migrate
  2. php artisan db:seed

5. Inicializar Proyecto de Esta Manera

  1. npm run build

Pasos para actualizar un proyecto a github

  1. git status -> ver el estado del proyecto
  2. git add . -> agregar todos los archivos
  3. git commit -m "mensaje" -> para guardar los cambios
  4. git push -> para subir el proyecto a github
  5. Esperar un tiempo para ver los cambios en github pages

Pasos para subir un proyecto a un repositorio de github

1. Inicializa el repositorio Git en tu proyecto:

git init

2. Configura el archivo .gitignore:

Asegúrate de que tu archivo .gitignore contenga lo siguiente para evitar subir archivos innecesarios:

/vendor
/node_modules
/.env
/.env.*
/public/storage
/storage/*.key
/public/hot
/public/build
/.phpunit.cache
.DS_Store
npm-debug.log
yarn-error.log

3. Añadir los archivos al repositorio Git:

git add .

4. Crear un commit:

git commit -m "Primer commit de mi proyecto Laravel"

5. Crear un repositorio en GitHub:

Ve a GitHub y crea un nuevo repositorio (sin inicializarlo con un README).

6. Conectar tu repositorio local con GitHub:

git remote add origin https://github.com/usuario/nombre-del-repositorio.git

7. Subir el proyecto a GitHub:

git push -u origin main
